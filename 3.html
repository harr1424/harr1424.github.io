This project involves creating a neural network for the purpose of classifying audio samples as expressing a predominant emotion (happy, sad, angry, etc). Neural networks in the context of machine learning are modeled after biological neural networks and function in a similar manner. Individual neurons, commonly called nodes, receive input as numerical data, process this data using an equation, and then output a result which can be fed as input to additional nodes or in the case of a terminal node can be used to make some prediction.
